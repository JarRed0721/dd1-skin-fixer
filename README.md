# DD1_Skin_Fixer

Many Darkest Dungeon skin Mods are facing the issue of aliasing, that makes those incredible art works hard to recoginze. 
This project is intended to reduce all the aliasing of the skin caused by the unfitting resolution of the textures.
The tool will resize the textures into apporiatie resolution, while adujusting any related file that is used in spine for proper 
funtctionailty of the mod.
Allowing any modders or players that want to fix this issue but don't know how or don't have time.
